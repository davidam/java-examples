Particle's Java Data Structures Dog 3D!

  You should really consider getting a Java capable browser... (or maybe just turning "java" on inside your browser...)
  Dog 3D!
 

Well, that's Dog 3D at it's best. It should be running rather nicely on most computers (if it's running a bit slow, download it, and decrease the applet size to 320x240; at that resolution, it should even work good on a 486!

To move, you can either use a mouse (by dragging it while clicked over the applet), or you can use your keyboard (the preferred method). To use the keyboard, click on the applet to make sure it has focus, and later move by pressing the arrow keys. Note that the keyboard method only seems to work in Windows, and not on UNIX platforms.

After you're done viewing it, click on the "back" button below to return to wherever you were; probably reading Java Data Structures Tutorial ;-)
