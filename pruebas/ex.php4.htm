Example Thing.java, module j714 (Java)
Example of destructor

 This is an example program from a Well House Consultants training course. Select here to jump to the module of the course on which this example is covered.

public class Thing {

public static int number_of_things = 0;
public String what;

public Thing (String what) {
	this.what = what;
	number_of_things++;
	}

protected void finalize ()  {
	number_of_things--;
	}

}
